# Video demo — siêu đơn giản, đọc theo từng dòng

Chuẩn bị: mở sẵn 2 cửa sổ trước khi bấm quay —
1. VS Code, mở sẵn file `server/src/index.ts`
2. Trình duyệt, mở sẵn `http://localhost:8082`

Bấm **Win + Alt + R** để bắt đầu quay (có tiếng mic), làm theo từng bước, đọc đúng câu trong ngoặc kép.

---

**Bước 1 — Giới thiệu (nhìn camera hoặc màn hình bất kỳ, 10 giây)**
Đọc: *"Đây là split-usdc, app chia tiền nhóm bằng USDC thật trên Arc."*

**Bước 2 — Code (chuyển sang cửa sổ VS Code, 20 giây)**
Đọc: *"Đây là code backend, dùng Circle Developer-Controlled Wallets SDK để tạo ví thật và gửi USDC thật trên Arc."*
→ Dùng chuột chỉ vào dòng có chữ `circleClient` và dòng có chữ `ARC-TESTNET` trong file đang mở, không cần giải thích thêm.

**Bước 3 — Demo app (chuyển sang trình duyệt, 60-90 giây)**
Đọc: *"Đây là app đang chạy thật."*
→ Bấm vào tab **Ví**, đọc: *"Đây là số dư USDC thật và lịch sử giao dịch thật."*
→ Vào 1 nhóm, bấm **Trả** trên 1 khoản chi, đọc: *"Bấm trả, đây là giao dịch USDC thật trên Arc."*
→ Đợi xem kết quả (thành công), đọc: *"Xong, đây là mã giao dịch thật."*

**Bước 4 — Kết thúc (10 giây)**
Đọc: *"Cảm ơn đã xem, link code trong đơn."*

Bấm **Win + Alt + R** để dừng quay. Video sẽ nằm trong thư mục **Videos > Captures**. Upload lên YouTube (chọn "Unlisted") hoặc Google Drive rồi copy link dán vào form.
